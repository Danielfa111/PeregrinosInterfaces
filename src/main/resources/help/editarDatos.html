Cómo editar tus datos

Volver

Instrucciones

  1. Introduce cambios en los campos con tus datos.
  2. Presiona confirmar cambios para editar tus datos correctamente

Descripción de los Botones

  • Confirmar cambios: Ejecuta el proceso para sellar el carnet del peregrino utilizando la opción seleccionada.
  • Cancelar: Permite anular la operación y volver al menú principal.

Problemas Frecuentes

  • Nombre incorrecto: Introduce un nombre valido
  • Usuario incorrecto Introduce un usuario que no exista ya
  • Correo incorrecto Asegurate de que el correo que introduces es valido
