Cómo sellar un carnet

Volver

Instrucciones

  1. Introduce el ID del Peregrino a sellar.
  2. Selecciona si el peregrino se va a alojar en la parada.
  3. Selecciona si la estancia del peregrino será VIP.

Descripción de los Botones

  • Sellar: Ejecuta el proceso para sellar el carnet del peregrino utilizando la opción seleccionada.
  • Cancelar: Permite anular la operación y volver al menú principal.

Problemas Frecuentes

  • ID incorrecto: Asegúrate de ingresar un ID válido. Verifica que no haya espacios ni caracteres erróneos.
  • Error al seleccionar la opción VIP: Revisa que la opción VIP no este seleccionada cuando la opcion de alojamiento no lo este
  • Problemas de conexión: Si la aplicación no responde, verifica tu conexión a la red y vuelve a intentarlo.
